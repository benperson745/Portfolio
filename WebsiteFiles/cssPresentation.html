CSS Gradient Presentation

The Problem

As a part of learning HTML, CSS, and Javascript, I created a presentation focused on adding in gradient CSS styling to a site that would allow a designer to show off video of their work. In my case, the aim of this styling was to allow for sizzle reels or trailers of my games to be shown off on my site home page.

My Tools

I quickly decided to make use of a simple Google Slides style presentation. This would allow for quick iteration of the presentation as I figured out the styling code. For the CSS, I experimented using Notepad++.

My Plan

I knew that I would not have the final footage of my in-progress projects to show off in the presentation. As a result, I was able to use some simple .gif files corresponding to a variety of games. This would let me hone in the scaling of the .gif in the boundaries of my CSS divs.

The Execution

When creating my CSS gradient, I at first struggled to get a consistent fade from one color to another. Once I had solved this issue, I switched out fading from color to color for fading from color to the .gif. As I went through this process, I documented each iteration of the CSS gradient in both images and code and began construction of my slides. By the time the CSS was complete, all that was required was to edit the slides and present.

The Result

In the end, I had a great piece of CSS styling which is viewable on the homepage of this site and the final presentation flowed without an issue due to solid documentation throughout the process as opposed to gathering the information after the fact.
